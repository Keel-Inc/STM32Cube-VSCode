# STM32Cube VS Code Example

A demonstration repository accompanying [Blame Your Tools](https://keel-inc.com/guides/blame-your-tools/).

## Build and Setup

Check out the repository
```powershell
git clone --recurse-submodules https://github.com/Keel-Inc/STM32Cube-VSCode.git
cd CBOR-RPC-DeviceControl
```

Build demo application
```
cmake --preset RelWithDebInfo
cmake --build build\RelWithDebInfo
```
